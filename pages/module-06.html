개요

학습 목표

단순 웹페이지를 넘어, 사용자 입력을 받고 결과를 보여주는 인터랙티브 웹앱을 만든다

이런 분께 추천합니다:

  • 단순 웹페이지를 넘어 동적 기능이 있는 도구를 만들고 싶은 분
  • 계산기나 변환 도구를 직접 만들어보고 싶은 분
  • 발표 자료를 새로운 방식으로 만들어보고 싶은 분

사례 6-1: 간단한 계산기/변환기 웹앱

Gemini 중급 20분

실무에서 자주 쓰는 다기능 단위 변환기를 웹앱으로 만듭니다.

Step 1: 다기능 변환기 만들기

Google AI Studio에 접속합니다.

프롬프트 
다기능 단위 변환기 웹앱을 만들어주세요.

변환 카테고리 (탭으로 전환):
1. 환율 계산기
   - 원화(KRW) ↔ 달러(USD), 엔(JPY), 유로(EUR)
   - 고정 환율 사용 (USD: 1350, JPY: 9.0, EUR: 1450)
   - 금액 입력 시 실시간 변환

2. 날짜 계산기
   - 두 날짜 사이 일수 계산
   - D-Day 계산 (특정 날짜까지 남은 일수)
   - 근무일 계산 (주말 제외)

3. 파일 크기 변환기
   - B, KB, MB, GB, TB 상호 변환
   - 입력 시 모든 단위로 동시 표시

디자인:
- 상단 탭 네비게이션
- 각 탭별 깔끔한 입력/출력 폼
- 입력 시 실시간 결과 표시 (버튼 클릭 불필요)
- 파란색 포인트 컬러, 둥근 모서리

Step 2: 추가 기능

프롬프트 
다음 기능을 추가해주세요:
1. 각 변환 결과 옆에 "복사" 버튼
2. 최근 변환 이력 (하단에 최근 5개 표시)
3. 키보드 Enter 키로 변환 실행
4. 결과 숫자에 천 단위 쉼표(,) 표시
5. 탭 전환 시 부드러운 애니메이션

사례 6-2: 인터랙티브 프레젠테이션 만들기

Claude 중급 25분

슬라이드 형태의 인터랙티브 프레젠테이션을 웹으로 만듭니다.

Step 1: 프레젠테이션 기본 구조

프롬프트 
인터랙티브 웹 프레젠테이션을 만들어주세요.

주제: "AI 시대의 디지털 정부혁신"

슬라이드 구성:
1. 표지 — 제목, 발표자 정보, NIA 로고 자리
2. 목차 — 클릭하면 해당 슬라이드로 이동
3. AI 기술 현황 — 주요 수치를 카운트업 애니메이션으로 표시
4. 디지털 정부혁신 방향 — 3개 카드 (클릭 시 상세 내용 펼침)
5. 사례 소개 — 이미지 placeholder + 설명
6. 기대 효과 — 차트 (Chart.js)
7. Q&A — "질문을 입력해주세요" 입력 폼

기능:
- 좌우 화살표 키로 슬라이드 이동
- 하단에 슬라이드 인디케이터 (점)
- 부드러운 슬라이드 전환 애니메이션
- 전체 화면 모드 버튼
- 슬라이드 번호 표시

디자인:
- 16:9 비율
- 어두운 남색 배경에 흰색 텍스트
- 슬라이드마다 다른 레이아웃

Step 2: 발표자 노트 추가

프롬프트 
발표자 모드를 추가해주세요:
1. "P" 키를 누르면 발표자 노트 패널이 토글
2. 각 슬라이드에 발표자 노트 (예: "이 슬라이드에서 3분 소요")
3. 발표 타이머 표시 (경과 시간)
4. 발표자 노트 패널에 다음 슬라이드 미리보기

사례 6-3: 엑셀 데이터를 보고서로 변환하기

Claude 중급 15분

CSV/엑셀 데이터를 입력하면 서식이 갖춰진 보고서 형태로 변환하는 도구를 만듭니다.

Step 1: 데이터-보고서 변환기

프롬프트 
CSV 데이터를 보고서 형태로 변환하는 웹 도구를 만들어주세요.

기능:
- CSV 데이터 입력 영역
- 보고서 템플릿 선택 (현황 보고서, 실적 보고서, 비교 분석)
- "변환" 버튼
- 보고서 출력: 표 + 차트 + 요약

예제 CSV:
부서,1분기,2분기,3분기,4분기
기획부,85,92,88,95
개발부,78,85,90,93
운영부,90,88,92,97
디자인부,82,87,85,91

보고서에 포함될 내용:
- 제목과 생성 날짜
- 데이터 테이블 (색상 포맷팅: 90 이상 초록, 80 미만 빨강)
- 분기별 추이 선 차트
- 부서별 비교 막대 차트
- 요약: 최고/최저 실적 부서, 전체 평균, 전분기 대비 변화

Chart.js 사용. 인쇄 가능한 디자인.

Step 2: 보고서 커스터마이징

프롬프트 
보고서 커스터마이징 옵션을 추가해주세요:
1. 보고서 제목 편집
2. 로고/기관명 입력
3. 표시할 차트 종류 체크박스 (선/막대/파이)
4. 색상 테마 선택 (기관용 파랑, 실적용 초록, 경고용 빨강)
5. "PDF 인쇄" 버튼 (브라우저 인쇄 기능 활용)

팁 & 트러블슈팅

  • 계산기/변환기는 자주 쓰는 도구를 만들어 즐겨찾기에 추가하세요
  • 프레젠테이션은 PPT 대신 사용할 수 있습니다. HTML 파일 하나로 어디서든 발표 가능합니다
  • 보고서 변환기는 매월 정기 보고서 작성 시 큰 도움이 됩니다

reveal.js 프레임워크를 사용하면 더 다양한 프레젠테이션 효과가 가능합니다.

심화 프롬프트: XLSX 파일 업로드 
SheetJS 라이브러리를 CDN으로 추가하고,
.xlsx 파일을 드래그 앤 드롭으로 업로드할 수 있게 해줘.
업로드된 파일을 파싱해서 자동으로 보고서를 생성해줘.

트러블슈팅

프레젠테이션 영역을 한 번 클릭하여 포커스를 맞춘 후 화살표 키를 누르세요. Artifacts 내에서는 키보드 이벤트가 제한될 수 있습니다. HTML 파일로 저장 후 브라우저에서 열어보세요.

Chart.js CDN이 로드되었는지 확인하세요. 인터넷 연결이 필요합니다. "Chart.js를 CDN 대신 인라인으로 포함해줘"라고 요청해볼 수 있습니다.

"인쇄 시 차트가 페이지 안에 맞도록 크기를 조정해줘"라고 요청하세요. @media print에서 차트 크기를 고정 크기로 설정하세요.
