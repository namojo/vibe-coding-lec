개요

학습 목표

이메일 분류, 일정 관리 등 반복 업무를 AI가 만든 도구로 자동화한다

이런 분께 추천합니다:

  • 매일 반복되는 이메일 정리에 시간을 뺏기는 분
  • 일정/업무 관리를 효율적으로 하고 싶은 분
  • 간단한 업무 도구를 직접 만들어보고 싶은 분

사례 5-1: 이메일 자동 분류/요약 도우미

Gemini 초급 25분

이메일 내용을 붙여넣으면 카테고리 분류와 요약을 자동 생성하는 도구를 만듭니다.

Step 1: 이메일 분류 도구 만들기

Gemini에 접속합니다.

프롬프트 
이메일 자동 분류 및 요약 도우미 웹 도구를 만들어주세요.

기능:
- 이메일 본문을 붙여넣는 텍스트 영역
- "분석하기" 버튼
- 분석 결과 표시:
  - 카테고리 (업무요청/보고/공지/일정/외부메일/기타)
  - 긴급도 (높음/보통/낮음) — 키워드 기반 판단
  - 요약 (핵심 내용 3줄)
  - 핵심 키워드 태그
  - 필요한 액션 (회신 필요/참고만/승인 필요 등)

분류 로직 (JavaScript 키워드 기반):
- "요청", "부탁", "처리" → 업무요청
- "보고", "결과", "현황" → 보고
- "공지", "안내", "알림" → 공지
- "회의", "일정", "미팅" → 일정
- "긴급", "급히", "오늘까지" → 긴급도 높음

예제 이메일을 기본으로 넣어주세요:
"안녕하세요. AI 활용 교육 관련 보고드립니다.
금주 교육 참석자 현황은 총 45명이며, 만족도 조사 결과
평균 4.2/5.0으로 집계되었습니다.
다음 주 금요일까지 최종 보고서 제출 부탁드립니다.
감사합니다."

깔끔한 카드형 디자인으로 만들어주세요.
예상 결과

이메일 입력 후 '분석하기'를 누르면 카테고리, 긴급도, 요약, 액션 아이템이 카드 형태로 표시됩니다.

Step 2: 응답 템플릿 기능

프롬프트 
다음 기능을 추가해주세요:

1. 분류 결과에 따른 "추천 응답 템플릿" 섹션
   - 업무요청 → "확인했습니다. [기한]까지 처리하겠습니다."
   - 보고 → "보고 잘 받았습니다. [피드백] 부분 확인 부탁드립니다."
   - 일정 → "일정 확인했습니다. 참석하겠습니다."
2. 템플릿의 [괄호] 부분을 직접 편집할 수 있게
3. "응답 복사" 버튼
4. 분석 이력 저장 (세션 내, 최근 10개)

사례 5-2: 일정 관리 자동화 스크립트

Claude 중급 25분

텍스트에서 일정 정보를 추출하고 캘린더 형태로 보여주는 도구를 만듭니다.

Step 1: 일정 추출 도구 만들기

프롬프트 
텍스트에서 일정을 자동 추출하여 캘린더에 표시하는 웹 도구를 만들어주세요.

기능:
- 텍스트 입력 영역 (이메일, 회의록 등 붙여넣기)
- "일정 추출" 버튼
- 추출된 일정 목록 (날짜, 시간, 내용, 참석자)
- 월간 캘린더 뷰에 일정 표시

일정 추출 로직 (JavaScript):
- "N월 N일", "N/N", "다음 주 월요일" 등 날짜 패턴 감지
- "회의", "미팅", "교육", "마감", "제출" 등 일정 키워드 감지
- 시간 패턴 ("오전 10시", "14:00" 등) 감지

예제 텍스트:
"3월 20일 오전 10시에 AI 활용 교육 회의가 있습니다.
참석: 김부장, 이과장.
교육 자료 제출 마감은 3월 25일입니다.
4월 1일~3일 교육 실시 예정이며,
4월 10일에 결과 보고 미팅을 진행합니다."

캘린더는 직접 구현하고 (테이블 기반),
일정이 있는 날짜에 색상 표시와 마우스 hover 시 상세 내용 표시.

Step 2: 캘린더 기능 강화

프롬프트 
캘린더를 개선해주세요:

1. 월 이동 (이전 달/다음 달) 버튼
2. 일정 클릭 시 상세 팝업 (내용, 시간, 참석자)
3. 일정별 색상 코딩 (회의=파랑, 마감=빨강, 교육=초록)
4. 오늘 날짜 강조 표시
5. 일정 직접 추가 기능 (날짜 클릭 → 입력 폼)

팁 & 트러블슈팅

  • 이메일 분류 도구를 즐겨찾기에 추가해두고, 중요 이메일을 빠르게 분석하는 데 활용하세요
  • 일정 추출 도구에 회의록을 붙여넣으면 다음 일정을 한눈에 파악할 수 있습니다
  • ICS 파일 내보내기를 사용하면 수동으로 캘린더에 입력하지 않아도 됩니다

Gmail API를 연동하면 이메일을 자동으로 가져와서 분류할 수 있습니다.

심화 프롬프트 
Gmail API를 사용해서 최근 10개 이메일을 가져오고,
자동으로 분류하는 기능을 추가해줘.
OAuth 2.0 인증 플로우도 포함해줘.

트러블슈팅

키워드 기반 로직의 한계입니다. 키워드를 추가하거나, "분류 키워드 목록을 더 확장해줘"라고 요청하세요. 실제 AI API를 연동하면 문맥 기반 분류가 가능합니다.

"다음 주 월요일" 같은 상대적 표현은 인식이 어려울 수 있습니다. "YYYY-MM-DD 형식도 인식하도록 정규표현식을 추가해줘"라고 요청하세요.

추출된 날짜가 현재 표시 중인 월과 맞는지 확인하세요. 월 이동 버튼으로 해당 월로 이동해보세요.
