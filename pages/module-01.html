학습 목표

Vibe Coding의 개념을 이해하고, AI 코딩 도구를 설정하고, 효과적인 프롬프트 작성법을 익힌다

이 모듈은 모든 학습자가 공통으로 이수하는 필수 모듈입니다. Vibe Coding이 무엇인지 이해하고, 이번 워크숍에서 사용할 3가지 AI 도구를 설정합니다.

Part 1: Vibe Coding이란?

"Vibe Coding"은 AI와 자연어 대화만으로 소프트웨어를 만드는 새로운 프로그래밍 방식입니다.

코드를 직접 작성하지 않습니다. 대신 AI에게 원하는 것을 말로 설명하면, AI가 코드를 생성하고 실행합니다.

"나는 코드를 보지도 않는다. 그냥 느낌대로(vibe) AI에게 말하고, 결과를 확인하고, 다시 말한다."

— Andrej Karpathy (OpenAI 공동창업자)

전통적 개발 vs Vibe Coding

구분 전통적 개발 Vibe Coding
입력코드 (Python, JavaScript 등)자연어 (한국어, 영어)
필요 지식프로그래밍 언어, 프레임워크원하는 결과물에 대한 명확한 설명
개발 속도수일~수주수분~수시간
대상개발자누구나
핵심 역량코딩 능력프롬프트 작성 능력

왜 지금 Vibe Coding인가?

  1. AI 코딩 도구의 급격한 발전: Claude, Gemini 등 AI가 생성하는 코드의 품질이 실무 수준에 도달
  2. 비개발자의 디지털 역량 강화: 아이디어만 있으면 누구나 프로토타입을 만들 수 있는 시대
  3. 업무 자동화 가속: 반복적인 업무를 AI와 함께 자동화하여 효율 극대화
  4. 정부/공공기관의 디지털 전환: NIA를 비롯한 공공기관에서 AI 활용 역량이 핵심 경쟁력

Vibe Coding으로 할 수 있는 것들

  • 회의록 자동 정리기
  • 데이터를 차트로 시각화
  • 나만의 웹페이지 만들기
  • 이메일 자동 분류/요약 도구
  • 설문조사 결과 분석기
  • 공공데이터 API 대시보드
  • Slack/Teams 알림봇
  • 실시간 통계 모니터링 페이지
  • 업무 자동화 파이프라인

Part 2: AI 도구 3종 소개 및 설정

Claude (Anthropic)

Claude.ai (웹 버전) — 비개발자 추천

  • 특징: 웹 브라우저에서 대화형으로 사용, Artifacts 기능으로 코드 결과를 실시간 미리보기
  • 강점: 장문 맥락 이해력 우수, 높은 코드 품질, 한국어 지원 뛰어남

설정 방법:

  1. claude.ai 접속
  2. "Sign up" → 이메일 또는 Google 계정으로 가입
  3. 이메일 인증 완료 → 바로 대화 시작

Claude Code (CLI 버전) — 개발자 추천

  • 특징: 터미널에서 직접 파일을 생성/수정/실행하는 고급 도구
# Node.js v18 이상 필요
npm install -g @anthropic-ai/claude-code
claude  # 안내에 따라 로그인

Antigravity (한국 AI 코딩 도구)

  • 특징: 한국어에 최적화된 AI 코딩 도구
  • 강점: 국내 서비스 연동 편리, 한국어 자연어 처리 우수

설정 방법:

  1. Antigravity 웹사이트 접속
  2. 계정 가입 또는 로그인
  3. "새 프로젝트" 생성 → 바로 사용

Gemini (Google)

  • Google AI Studio: 웹 기반 프로토타이핑, 다양한 모델 선택 가능
  • Gemini: 간편 대화형 인터페이스
  • 강점: Google Workspace 연동, 멀티모달(텍스트+이미지) 지원

설정 방법:

  1. aistudio.google.com 또는 gemini.google.com 접속
  2. Google 계정으로 로그인 → 바로 사용

Part 3: 효과적인 프롬프트 작성법

AI에게 잘 요청하는 것이 Vibe Coding의 핵심 역량입니다.

원칙 1: 구체적으로 설명하라

나쁜 예 좋은 예
"웹페이지 만들어줘""파란색 헤더가 있고, 3개의 카드로 팀원을 소개하는 웹페이지를 만들어줘"
"예쁘게 해줘""배경은 흰색, 포인트 컬러는 #3B82F6, 둥근 모서리의 카드 디자인으로 해줘"
"데이터 분석해줘""이 CSV 파일의 월별 매출 데이터를 막대 차트로 시각화해줘"

원칙 2: 맥락을 제공하라

좋은 프롬프트 예시 
나는 NIA에서 디지털혁신 업무를 담당하고 있어.
매주 회의록을 정리하는데 시간이 많이 걸려.
회의 내용을 붙여넣으면 자동으로 정리해주는 도구를 만들고 싶어.
참석자, 논의 사항, 결정 사항, 다음 할 일로 구분해서 정리해줘.

원칙 3: 단계적으로 요청하라

한 번에 모든 것을 요구하지 마세요. 작게 시작하고 점진적으로 기능을 추가하세요.

단계적 요청 예시 
[1단계] "간단한 회의록 정리 웹페이지를 만들어줘"
[2단계] "입력란에 회의 내용을 붙여넣으면 자동으로 구분해서 정리해줘"
[3단계] "정리된 결과를 복사할 수 있는 버튼을 추가해줘"
[4단계] "PDF로 다운로드할 수 있는 기능도 넣어줘"

원칙 4: 피드백으로 개선하라

AI의 결과물을 확인하고, 마음에 들지 않는 부분을 구체적으로 지적하세요.

피드백 예시 
전체적으로 좋은데, 다음 부분을 수정해줘:
1. 헤더 글자가 너무 작아 — 24px로 키워줘
2. 카드 사이 간격이 좁아 — 20px 정도로 넓혀줘
3. 모바일에서 카드가 가로로 나열됨 — 세로로 쌓이게 해줘

트랙별 팁

  • Claude.ai를 먼저 시작하세요: Artifacts 미리보기가 있어서 결과를 가장 쉽게 확인할 수 있습니다
  • 두려워하지 마세요: AI가 만든 코드를 이해할 필요는 없습니다. 결과가 원하는 대로 나오는지만 확인하세요
  • 실험하세요: 같은 요청을 여러 도구에 해보고, 결과를 비교해 보세요
  • Claude Code는 기존 프로젝트 폴더에서 실행하면 프로젝트 컨텍스트를 자동으로 파악합니다
  • 각 도구의 API를 활용하면 자동화 파이프라인을 구축할 수 있습니다
  • Claude Code + Gemini Code Assist를 IDE에서 동시에 활용하는 것도 가능합니다

트러블슈팅

  • Claude: 일부 국가에서 접속이 제한될 수 있습니다
  • Gemini: Google 계정이 필요합니다. 회사 Google Workspace 계정으로도 가능합니다
