사례 2-1: 나만의 포트폴리오 웹페이지 만들기

Claude 초급 30분

Claude와 대화만으로 개인 포트폴리오 웹페이지를 완성합니다. 완성 결과에는 프로필 영역, 자기소개, 관심 분야 뱃지, 주요 업무 목록, 연락처가 포함됩니다.

Step 1: 기본 페이지 만들기

Claude.ai에 접속하여 다음 프롬프트를 입력합니다.

프롬프트 
자기소개 웹페이지를 만들어주세요.

포함할 내용:
- 이름: 홍길동
- 소속: NIA 디지털혁신부
- 직함: 선임연구원
- 소개: "디지털 정부혁신과 AI 활용 연구를 담당하고 있습니다."
- 관심 분야: AI, 디지털 전환, 공공데이터

디자인 요구사항:
- 깔끔하고 모던한 디자인
- 흰색 배경에 파란색 포인트 컬러
- 프로필 사진 자리에 원형 placeholder
- 모바일에서도 잘 보이게

HTML, CSS를 하나의 파일에 포함해주세요.
예상 결과

Claude가 완성된 HTML/CSS 코드를 생성합니다. Artifacts 패널에서 웹페이지 미리보기가 바로 나타납니다.

Step 2: 디자인 개선하기

프롬프트 
이 페이지를 더 세련되게 개선해주세요:

1. 상단에 은은한 그라데이션 배경 (파란색 → 보라색 계열)
2. 프로필 카드에 그림자 효과 추가
3. 관심 분야를 태그 형태의 뱃지로 표시
4. 마우스를 올리면 카드가 살짝 떠오르는 hover 효과
5. 페이지가 열릴 때 부드럽게 나타나는 fade-in 애니메이션
예상 결과

그라데이션 배경, 카드 그림자, 뱃지 태그, hover 효과, fade-in 애니메이션이 적용됩니다.

Step 3: 섹션 추가하기

프롬프트 
다음 섹션들을 추가해주세요:

1. "주요 업무" 섹션
   - AI 정책 연구 및 분석
   - 디지털 정부혁신 프로젝트 관리
   - 공공데이터 활용 촉진

2. "연락처" 섹션
   - 이메일: hong@nia.or.kr
   - 전화: 02-1234-5678
   - 위치: 세종특별자치시

3. 하단 footer에 저작권 표시: "© 2026 홍길동. All rights reserved."

각 섹션은 아이콘(이모지)으로 시각적으로 구분해주세요.

Step 4: 나만의 정보로 커스터마이징

예제 정보를 본인의 실제 정보로 교체합니다.

프롬프트 
다음 정보로 내용을 변경해주세요:

- 이름: [본인 이름]
- 소속: [본인 부서]
- 직함: [본인 직함]
- 소개: [본인 한 줄 소개]
- 관심 분야: [본인 관심 분야 3~5개]
- 주요 업무: [본인 주요 업무 3개]
- 이메일: [본인 이메일]

Step 5: 결과물 저장하기

Artifacts에서 복사 (비개발자 추천)

  1. Artifacts 패널에서 "Copy" 버튼 클릭
  2. 메모장 또는 텍스트 편집기 열기
  3. 붙여넣기 (Ctrl+V / Cmd+V)
  4. "다른 이름으로 저장" → 파일명: index.html → 파일 형식: "모든 파일"
  5. 저장한 파일을 더블클릭하면 브라우저에서 열립니다

Claude Code 사용

mkdir my-profile && cd my-profile
claude
# "자기소개 웹페이지를 index.html로 만들어줘"

GitHub Pages로 배포하기

git init
git add index.html
git commit -m "나만의 프로필 페이지"
git remote add origin https://github.com/[username]/[username].github.io.git
git push -u origin main

배포 후 https://[username].github.io에서 확인 가능합니다.

사례 2-2: 팀 소개 페이지 만들기

Antigravity 초급 20분

Antigravity로 부서/팀 소개 페이지를 만듭니다.

Step 1: 기본 팀 소개 페이지 만들기

Antigravity에 접속하여 새 프로젝트를 생성합니다.

프롬프트 
NIA 디지털혁신부 팀 소개 웹페이지를 만들어주세요.

포함할 내용:
- 부서명: 디지털혁신부
- 부서 소개: "디지털 정부혁신과 지능정보사회 구현을 위한 핵심 부서입니다"
- 팀원 3명 (카드 형태):
  - 김철수 / 부장 / AI 정책
  - 이영희 / 과장 / 데이터 분석
  - 박민수 / 대리 / 서비스 기획
- 주요 사업: 디지털 정부혁신, AI 활용 확산, 공공데이터 개방

디자인:
- 깔끔한 화이트 테마
- 팀원 카드에 hover 효과
- 모바일 반응형

Step 2: 디자인 커스터마이징

프롬프트 
다음을 개선해주세요:
1. 상단에 NIA 로고 자리 placeholder 추가
2. 팀원 카드에 프로필 이미지 placeholder (원형, 회색 배경)
3. 주요 사업을 아이콘과 함께 그리드 형태로 배치
4. 하단에 연락처와 오시는 길 섹션 추가

Step 3: 인터랙티브 요소 추가

프롬프트 
인터랙티브 요소를 추가해주세요:
1. 팀원 카드를 클릭하면 상세 정보가 펼쳐지는 효과
2. 주요 사업 섹션에 탭으로 각 사업 상세 내용 전환
3. 스크롤 시 각 섹션이 순차적으로 나타나는 애니메이션

팁 & 트러블슈팅

프롬프트 작성 요령

  • 구체적으로: "예쁘게"보다 "파란색 배경에 흰색 글자로"가 훨씬 좋습니다
  • 나눠서 요청: 한 번에 모든 걸 요구하지 말고, 단계별로 추가해 나가세요
  • 참고 이미지 설명: "네이버 프로필 페이지 같은 느낌으로"처럼 참고 대상을 언급하세요

결과물이 마음에 안 들 때

  • "글자 크기를 20% 더 키워주세요" — 구체적 수치 제공
  • "이 부분은 마음에 드는데, 배경색만 바꿔주세요" — 좋은 부분 명시
  • "처음부터 다시 만들어주세요. 미니멀한 스타일로" — 완전히 새로 시작

반응형 + 다크모드 강화

심화 프롬프트 
반응형 디자인과 다크 모드를 추가해줘:
- 768px 이하에서는 단일 컬럼 레이아웃
- prefers-color-scheme으로 다크 모드 자동 전환
- 다크 모드 수동 토글 버튼도 추가

트러블슈팅

"이 코드의 미리보기를 보여주세요"라고 요청해보세요. 브라우저를 새로고침한 후 다시 시도하세요.

파일 저장 시 인코딩을 UTF-8로 설정하세요. 코드 상단에 <meta charset="UTF-8">이 있는지 확인하세요.

같은 대화창 안에서 계속 요청하세요. 새 대화를 시작하면 이전 맥락이 사라집니다.

파일 확장자가 .html인지 확인 (.txt가 아닌지). 파일을 마우스 오른쪽 클릭 → "연결 프로그램" → Chrome/Edge 선택
