# 모듈 3: 문서 자동화

## 실습 사례

### 사례 3-1: 회의록 자동 정리기 (meeting-minutes.html)
- 회의 내용을 붙여넣으면 구조화된 회의록으로 자동 정리
- 키워드 기반 파싱 (참석, 결정, 할 일 등)
- 마크다운 복사, 초기화, 예제 데이터 지원
- 사용 도구: Claude / Gemini / Antigravity

### 사례 3-2: 정책 브리핑 자동 생성기 (policy-briefing.html)
- 정책 정보를 입력하면 공문서 스타일의 브리핑 문서 자동 생성
- A4 인쇄 최적화, 인쇄 기능 내장
- 사용 도구: Claude / Gemini / Antigravity

### 사례 3-3: 다국어 문서 번역 도우미 (translator.html)
- 좌우 레이아웃의 번역 인터페이스
- 언어 선택, 번역 스타일, 글자수 카운터, 히스토리
- 사용 도구: Claude / Gemini / Antigravity

## 실행 방법

HTML 파일을 브라우저에서 직접 열어 확인합니다.

```
open meeting-minutes.html
open policy-briefing.html
open translator.html
```
