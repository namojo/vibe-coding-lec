# 모듈 6: 웹 애플리케이션

AI를 활용하여 실용적인 웹 애플리케이션을 만드는 실습입니다.

## 실습 파일

### 사례 6-1: 다기능 단위 변환기 (`unit-converter.html`)
환율, 날짜, 파일 크기를 변환하는 다기능 도구입니다.

**주요 기능:**
- 3개 탭: 환율 계산기, 날짜 계산기, 파일 크기 변환기
- 환율 계산: 6개 통화(KRW, USD, EUR, JPY, CNY, GBP) 상호 변환
- 날짜 계산: 두 날짜 사이 기간, 날짜에 일수 더하기/빼기
- 파일 크기: B ~ PB 단위 자동 변환
- 실시간 변환 (입력 즉시 결과)
- 결과 복사 버튼
- 변환 이력 저장 (localStorage)
- 천단위 쉼표 자동 포맷

### 사례 6-2: 인터랙티브 프레젠테이션 (`presentation.html`)
"AI 시대의 디지털 정부혁신" 주제의 7장 슬라이드 프레젠테이션입니다.

**주요 기능:**
- 7장 슬라이드 구성: 표지, 현황, AI 적용 분야, 로드맵, 예산, 기대효과, 마무리
- 좌우 화살표 키 이동
- 하단 인디케이터 (도트 네비게이션)
- 카운트업 애니메이션 (숫자 동적 표현)
- 펼침 카드 (클릭 시 상세 내용 표시)
- Chart.js 차트 (예산 투자 현황)
- 발표자 노트 (P키 토글)
- 전체화면 (F키)
- 어두운 남색(#0f172a) 배경, 16:9 비율 대응

### 사례 6-3: 데이터 보고서 변환기 (`data-to-report.html`)
CSV 데이터를 전문적인 보고서 형태로 자동 변환합니다.

**주요 기능:**
- CSV 데이터 입력 영역
- 3가지 보고서 템플릿: 현황 보고서, 실적 보고서, 비교분석 보고서
- 자동 생성 요소: 요약 카드, 데이터 표, 차트, 분석 텍스트
- 색상 포맷팅: 90 이상 초록, 80 미만 빨강
- 달성률, 증감률 자동 분석
- Chart.js 차트 (막대, 도넛/선)
- 인쇄 / PDF 저장 기능

**사용 라이브러리:** Chart.js (CDN)

## 실행 방법
각 HTML 파일을 브라우저에서 직접 열어 실행합니다.
