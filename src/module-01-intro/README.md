# 모듈 1: Vibe Coding 소개

## 모듈 설명

이 모듈은 Vibe Coding의 개념과 AI 코딩 도구의 활용법을 소개하는 이론 중심 모듈입니다.

### 다루는 내용

- **Vibe Coding이란?** — 자연어로 소프트웨어를 만드는 새로운 개발 패러다임
- **AI 도구 3종 비교** — 각 도구의 특성과 적합한 사용 사례
  - Claude (Anthropic): 정교한 코드 생성, 맥락 이해력 우수
  - Gemini (Google): 멀티모달 지원, Google 생태계 연동
  - Antigravity: 한국어 특화, 국내 업무 환경 최적화
- **효과적인 프롬프트 작성법** — 원하는 결과를 얻기 위한 지시문 구성 전략
