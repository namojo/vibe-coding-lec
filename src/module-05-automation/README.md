# 모듈 5: 업무 자동화

AI를 활용하여 반복적인 업무를 자동화하는 도구를 만드는 실습입니다.

## 실습 파일

### 사례 5-1: 이메일 자동 분류/요약 도우미 (`email-classifier.html`)
이메일을 자동으로 분류하고 요약하는 도구입니다.

**주요 기능:**
- 이메일 본문 입력 및 예제 이메일 5종 내장
- 카테고리 자동 분류: 업무요청 / 보고 / 공지 / 일정 / 외부메일
- 긴급도 판단 (빨강/노랑/초록)
- 본문 요약 및 핵심 키워드 태그 추출
- 필요 액션 아이템 목록
- 추천 응답 템플릿 (편집 및 복사 가능)
- 분석 이력 저장 (localStorage)

### 사례 5-2: 일정 추출 및 캘린더 (`schedule-extractor.html`)
텍스트에서 날짜와 일정을 자동으로 추출하여 캘린더에 표시합니다.

**주요 기능:**
- 자유 형식 텍스트에서 날짜/일정 자동 추출
- 월간 캘린더 뷰 (테이블 기반)
- 일정별 색상 코딩: 회의(파랑), 마감(빨강), 교육(초록), 기타(노랑)
- 월 이동 및 오늘 날짜 강조
- 일정 클릭 시 상세 정보 팝업
- ICS 파일 내보내기

## 실행 방법
각 HTML 파일을 브라우저에서 직접 열어 실행합니다.
