# 모듈 9: 종합 프로젝트

## 사례 9-1: React 업무 관리 대시보드 (프롬프트 가이드)
**파일:** `setup-guide.md`

React 기반 칸반 보드 대시보드를 AI 프롬프트로 단계별 구축하는 가이드입니다.
실제 코드가 아닌 **프롬프트 설계 방법론**을 다룹니다.

### 가이드 내용
- 8단계 순차 프롬프트 (초기설정 → 데이터 → 레이아웃 → 칸반 → 폼 → 필터 → 통계 → 마무리)
- 좋은 프롬프트 작성 팁
- 반복 개선 및 디버깅 프롬프트 예시
- 완성 프로젝트 구조 참고

## 사례 9-2: NIA 업무 도우미 챗봇
**파일:** `chatbot.html`

메신저 앱 형태의 키워드 기반 업무 도우미 챗봇입니다.

### 주요 기능
- 메신저 UI (최대 450px, 모바일 최적화)
- 말풍선 대화 (봇=왼쪽 파란색, 사용자=오른쪽 회색)
- 키워드 매칭 응답: 회의실, 휴가, 식당, 연락처, IT지원, 일정
- 빠른 답변 버튼 (상황별 추천 질문)
- 입력중... 애니메이션 표시
- 대화 히스토리 localStorage 자동 저장
- 대화 내보내기 (텍스트 파일 다운로드)
- 응답 피드백 수집 (도움됨/아쉬움)

### 실행 방법
```
브라우저에서 chatbot.html 파일을 직접 열기
```

## 사례 9-3: 실시간 시스템 모니터링 대시보드
**파일:** `monitoring-dashboard.html`

실시간 서버 상태를 시뮬레이션하는 다크 테마 모니터링 대시보드입니다.

### 주요 기능
- 다크 테마 전문 대시보드 UI
- 실시간 메트릭: 접속자 수, 서버 상태, API 응답시간, CPU 사용률
- 6대 서버 상태 표시 (정상/주의/장애)
- API 응답시간 실시간 차트 (최근 60초)
- 접속자 수 추이 차트 (최근 60초)
- 매초 데이터 갱신 (setInterval 시뮬레이션)
- 간헐적 장애 시뮬레이션 및 경고 배너
- 이벤트 이력 테이블
- 시뮬레이션 속도 조절 (0.5x ~ 5x)
- 일시정지/재개 기능

### 실행 방법
```
브라우저에서 monitoring-dashboard.html 파일을 직접 열기
```

## 사용 기술
- HTML5, CSS3, Vanilla JavaScript
- Chart.js (CDN)
- localStorage API
