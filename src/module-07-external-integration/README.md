# 모듈 7: 외부 데이터 연동 대시보드

## 사례 7-1: Google Sheets 민원 현황 대시보드
**파일:** `sheets-dashboard.html`

Google Sheets에서 가져온 민원 데이터를 시각화하는 대시보드입니다.
(데모용으로 하드코딩된 데이터 사용)

### 주요 기능
- 월별 민원 접수/처리 현황 요약 카드 (전월대비 증감 표시)
- 복합 차트: 막대(접수 건수) + 선(처리율) 차트
- 부서별 민원 비중 도넛 차트
- 정렬 가능한 상세 테이블 (상태 배지 포함)
- 30초 주기 자동 갱신 시뮬레이션
- 처리율 미달 시 경고 배너 표시
- CSV 보고서 내보내기

### 실행 방법
```
브라우저에서 sheets-dashboard.html 파일을 직접 열기
```

## 사례 7-2: 공공데이터 인구통계 대시보드
**파일:** `public-data-dashboard.html`

공공데이터포털의 인구통계 데이터를 활용한 대시보드입니다.

### 주요 기능
- 6개 광역시 인구 데이터 (서울/부산/대구/인천/대전/세종)
- 요약 카드 (총인구, 평균 고령화율, 세대수, 인구증가율)
- 인구 현황 막대 차트 (방사형 전환 가능)
- 고령화율 수평 막대 차트 (색상 구분)
- 멀티 선택 지역 필터 (태그 방식)
- 정렬 옵션 (인구순/고령화율순)
- 지역 선택 → 8개 항목 상세 카드
- 2개 지역 나란히 비교 모드

### 실행 방법
```
브라우저에서 public-data-dashboard.html 파일을 직접 열기
```

---

## 사용 기술
- Chart.js (CDN)
- Vanilla JavaScript
- CSS Grid / Flexbox
