# 모듈 2: 나의 첫 웹페이지

## 실습 사례

### 사례 2-1: 개인 포트폴리오 (portfolio.html)
- AI에게 자연어로 요청하여 개인 포트폴리오 웹페이지를 생성
- 프로필, 관심분야 태그, 주요 업무, 연락처 등 포함
- 그라데이션 배경, 카드 그림자, 애니메이션 등 모던 디자인 적용
- 사용 도구: Claude / Gemini / Antigravity

### 사례 2-2: 팀 소개 페이지 (team-page.html)
- 부서 소개와 팀원 카드로 구성된 팀 페이지 생성
- 팀원 클릭 시 상세 정보 표시, 스크롤 애니메이션
- 사용 도구: Claude / Gemini / Antigravity

## 실행 방법

HTML 파일을 브라우저에서 직접 열어 확인합니다.

```
# 방법 1: 파일 더블클릭
# 방법 2: 터미널에서 열기
open portfolio.html
open team-page.html
```
