# 모듈 4: 데이터 시각화

AI를 활용하여 데이터를 시각적 차트와 대시보드로 변환하는 실습입니다.

## 실습 파일

### 사례 4-1: CSV 데이터 차트 시각화 도구 (`chart-maker.html`)
CSV 형식의 데이터를 다양한 차트로 시각화하는 도구입니다.

**주요 기능:**
- CSV 텍스트 입력 (월별 방문자수/가입자수 예제 데이터 내장)
- 차트 유형 선택: 막대, 선, 파이
- 차트 제목 입력
- 4가지 색상 테마: 기본 / 파스텔 / 비비드 / 모노톤
- PNG 파일 저장
- 인터랙티브 툴팁

**사용 라이브러리:** Chart.js (CDN)

### 사례 4-2: 설문조사 결과 분석 대시보드 (`survey-analyzer.html`)
설문조사 결과를 종합 분석하는 대시보드입니다.

**주요 기능:**
- 고객 만족도 조사 예제 데이터 내장
- 요약 카드: 총 응답자, 평균 만족도, 만족 비율, 최고 항목
- 문항별 수평 막대 차트 (스택)
- 전체 만족도 도넛 차트
- 항목별 레이더 차트
- 슬라이드 데이터 편집 패널
- 분석 보고서 자동 생성
- 인쇄 기능

**사용 라이브러리:** Chart.js (CDN)

## 실행 방법
각 HTML 파일을 브라우저에서 직접 열어 실행합니다.
